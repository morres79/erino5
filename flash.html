Установочная Флэшка

Делаем флешку загрузочной, при помощи загрузчика Grub4Dos.
  1. Скачиваем набор утилит для подготовки флешки grub4dos.zip
  2. Распаковываем grub4dos.zip на свой компьютер, например, на диск C:, тогда полный путь к утилитам подготовки флешки будет «C:\grub4dos».
  3. Вставляем флешку объёмом не менее 8 Гб (рекомендуется 32 Гб) в ПК и запускаем программу «C:\grub4dos\HPUSBFW.EXE»:



    в списке «Device» выбираем нужную флешку, «File System» – NTFS, «Volume label» – что угодно, а так же лучше отметить галочку «Quick Format» чтобы долго не ждать. Жмём кнопку «Start», дожидаемся окончания процесса, жмём «Ok», закрываем эту программу.
  4. Запускаем «C:\grub4dos\grubinst_gui.exe»:
    ставим точку напротив «Disk» и из списка выбираем флешку (скорее всего она последняя в списке, ориентируемся по объему).



    Тут надо быть осторожным и не ошибиться в выборе, иначе, можно затереть загрузчик на основном диске ПК, и он перестанет грузиться.
    Не нужно отмечать никакие галочки, просто нажимаем кнопку «Install».

Флешка теперь пригодна для того чтоб с неё можно было загружаться. Переходим к следующему разделу чтоб сделать её установочной.

Делаем флешку установочной.

Скачиваем архивы, которые надо будет распаковать на флешку (ВНИМАНИЕ! Все архвы распаковываем в корень флешки с ЗАМЕНОЙ файлов в той последовательности, в которой они тут указаны!):

  1. flash_skeleton.zip (690 Мб) - каркас самой флешки без установочных образов, дистрибутивов и дополнений.
  2. Образы установки Windows:
    1. flash2a_images_w7.zip (5220 Мб) - образ установки Windows 7,
    2. flash2b_images_w8.zip (3829 Мб) - образ установки Windows 8.1,
    3. flash2c_images_w10.zip (3117 Мб) - образ установки Windows 10,
    4. flash2d_images_wxp.zip (0 Мб) - образы установки Windows XP. Временно недоступен.
  3. flash3_addon_base.zip (130 Мб) - включает установку базовых программ вслед за установкой Windows.
    1. flash3a_distr_of10.zip (1529 Мб) - добавляет установку Office 2010.
    2. flash3b_distr_of13.zip (1233 Мб) - добавляет установку Office 2013. Не устанавливается в Windows XP
    3. flash3c_distr_of16.zip (637 Мб) - добавляет установку Office 2016. Не устанавливается в Windows XP
    4. flash3d_distr_updatepack.zip (832 Мб) - добавляет установку пакета обновлений на сентябрь 2016. Только для Windows 7 и XP
  4. flash4_addon_drv.zip (219 Мб) - набор распространённых драйверов - ОС во время установки может их "подхватить".
  5. flash5_addon_iso.zip (1853 Мб) - набор очень важных утилит (ERD, MemTest, MHDD, Acronis и т.п.).
  6. flash6_addon_live.zip (1853 Мб) - добавляет возможность загружаться в Ubuntu Live.

P.S. Для всех возможностей необходима флешка не менее 32 Гб. Каждый последующий пункт добавляет флешке функционал. Если, например, распаковать последний архив flash6_addon_live.zip, пропустив все предыдущие (кроме 1го, конечно же), то появятся все пункты в меню Grub4Dos, но работать они не будут.

Установка Windows будет работать только если выполнить хотя бы один из подпунктов 2го пункта. Если, например, пропустить пункты 2.a и 2.b, а сразу распаковать flash2c_images_w10.zip (пункт 2.с), то в меню Total Commander появятся и Windows 7 и Windows 8.1, но работать они не будут, а Windows 10 будет устанавливаться нормально.

Добавление офисов или набора обновлений может быть сделано в любом сочетании и в любой последовательности. Программа пост-установки сначала проверяет наличие установочного файла и, если его не находит, то и не показывает эту программу в списке приложений для автоматической установки.
